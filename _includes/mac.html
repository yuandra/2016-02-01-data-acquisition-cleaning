Mac

Please go through all the installation steps below and make sure that you not only installed them, but start them up to make sure they're working. If you have any problems, don't hesitate to email the instructors to ask for help, or arrive early on the first day of the workshop to get help.

  1. A web browser
    For this workshop you will need to have a web browser. Many people already have Microsoft Edge / Internet explorer but for this workshop we will use Google Chrome as our web browser
    If you don't have Google Chrome, then here are the install instructions for Google Chrome :
    • Download the Installer
      Install Google Chrome by going to the installation page.Click on download to download the installer.
    • Install Google Chrome
      Once the installer is downloaded, double click on it and it should install.
    • To use Google Chrome, double click on the icon and it will open.

  2. A spreadsheet program
    For this workshop you will need a spreadsheet program. Many people already have Microsoft Excel installed, and if you do, you're set!
    If you need a spreadsheet program, there are a few other options, like OpenOffice and LibreOffice. Install instructions for LibreOffice, which is free and open source, are here.
    • Download the Installer
      Install LibreOffice by going to the installation page. The version for Mac should automatically be selected. Click Download Version 5.0.4. You will go to a page that asks about a donation, but you don't need to make one. Your download should begin automatically.
    • Install LibreOffice
      Once the installer is downloaded, double click on it and it should install.
    • To use LibreOffice, double click on the icon and it will open.

  3. OpenRefine
    OpenRefine (previously Google Refine) is a tool for data cleaning that runs through a web browser, and any browser - Safari, Firefox, Chrome, Explorer - should work fine. You will need to download Google Refine and install it, and when you open it, it will run through the browser, but you don't need an internet connection, and the data will all be stored on your computer.
    • Install Java
    • To use OpenRefine you will need to have a JDK installed. Go to Java Download Page.
    • Click on the mac version to download the install file
    • Once downloaded, double click on the installer to install

    • Install Open Refine
    • Go to the OpenRefine Github download page
    • Click on openrefine-mac-2.6-rc.2.dmg to download the install file
    • Open the downloaded .dmg file
    • Drag the icon in to the Applications folder
    • Double click on the icon and OpenRefine Refine will then open in your web browser.
    • If it doesn't open automatically, open a web browser after you've started the program and go to the URL http://localhost:3333 and you should see OpenRefine.
