DIT4C

For the Open Refine, besides instaling in your own computer, you can use a platform called DIT4C (Data Intensive Tools for the Cloud)

To use DIT4C, you will need to have any of these 3 account.

  • Australia Access Federation (AAF) account. If you have a *.edu.au from an Australian Universities you usually have it
  • Github account. Github is a version control + code sharing site that is widely used for the open source community. You can make an account by going http://github.com
  • Twitter account. Twitter is a social media site where you can post your short thoughts. You can make an account by going http://twitter.com
After you have either one of AAF, Github, and twitter account, then you can access DIT4C.

  • Access DIT4C by accessing http://resbaz.cloud.edu.au
  • Login to DIT4C using either one of your account
  • The main process you will use while working in the DIT4C is that you will "claim" a computer node and then you will start a "container" that has your program you want to run inside in the computer node.
  • Click on the computer-node tab in the upper left
  • To claim a compute node, you will be given a "share code" that you can input to claim a compute node. Each compute node will have its own share code. The share will be given to you during the workshop.
  • Click on the claim computer-node access in the upper left
  • Select the compute node according to the share code you have been given and click claim. The compute node name should between res resbaz-mel-openrefine-1/2/3
  • After you claim a compute node, then you will have to start the container inside the computer node that you have claimed
  • Click on the add container in the upper right
  • Select the compute node that you just claimed (resbaz-mel-openrefine-1/2/3).
  • Select the image "Open Refine"
  • Give the container an unique name
  • Click on start to get the container start
  • You will see that the container will be shown on the list. Click on the name of your container to open the container window.
  • You will see a blue background with Open Refine logo. Click on the Open Refine logo to start Open Refine
  • Congratulations ! You have run open refine in the DIT4C !
